🎯 BoLuo 图片压缩 - Blob URL 使用示例

💡 关键概念：Blob URL

当你使用 BoLuo.compressToBlob() 获得压缩后的 Blob 时，需要通过 URL.createObjectURL(blob) 创建一个临时的 URL 来使用它。

📁 选择图片文件体验 Blob URL 的使用

📝 核心代码示例：

// 1. 压缩图片获取 Blob
const compressedBlob = await BoLuo.compressToBlob(file, { quality: 80 });

// 2. 创建 Blob URL
const blobUrl = URL.createObjectURL(compressedBlob);

// 3. 使用 Blob URL
// 显示图片
img.src = blobUrl;

// 下载文件
link.href = blobUrl;
link.download = 'compressed.jpg';

// 上传到服务器
const formData = new FormData();
formData.append('image', compressedBlob, 'compressed.jpg');

// 4. 清理资源（可选）
URL.revokeObjectURL(blobUrl);

🔄 完整的使用流程：

  1. 压缩：使用 BoLuo.compressToBlob() 获取压缩后的 Blob
  2. 创建URL：使用 URL.createObjectURL(blob) 创建临时URL
  3. 使用URL：将URL用于图片显示、下载链接或其他用途
  4. 清理资源：使用完后调用 URL.revokeObjectURL(url) 释放内存
